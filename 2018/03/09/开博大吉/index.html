使用hexo+github写博客

hexo简介

理解为使用Markdown语法写文档

##步骤

  1. cd ~/Desktop 进入桌面
  2. mkdir myblog 创建一个文件，之后博客的文件都放在这里面
  3. sudo npm install -g hexo-cli 安装hexo
  4. npm i 作用不明
  5. hexo new 开博大吉 新建一篇标题为开博大吉的博客
  6. xdg-open ~/Desktop/myblog/source/_posts/开博大吉.md 打开.md文件开始编辑博客
